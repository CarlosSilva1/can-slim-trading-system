## 📊 Sistema Híbrido de Trading Baseado em CAN SLIM (IBD/O'Neil)

Sistema institucional que combina **ranking quantitativo de liderança** com **execução técnica disciplinada**.

### 🧠 Camada 1: Python Engine
- ✅ RS Rating (1-99)
- ✅ Market Pulse (UPTREND/NEUTRAL/DOWNTREND)
- ✅ Pivot Detection
- ✅ Breakout Validation

### 📈 Camada 2: TradingView
- ✅ Confirmação Visual
- ✅ Alertas Automáticos

## 🚀 Quick Start

```bash
cd python-engine
pip install -r requirements.txt
python src/scanner.py
```
